# Territory Foods Code Challenge

## Install
> ``` git clone...``` repository 
> open 2 different terminal windows, both into ROOT directory

### Terminal 1

``` cd backend && yarn ```

``` yarn start```

This will open http://localhost:3333

### Terminal 2

``` cd client && yarn ```

``` yarn start```

This will open http://localhost:3000
